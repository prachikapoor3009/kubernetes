- Kubernetes has a control plane and a **data plane**.

**Components of control plane**
- API Server
- etcd
- Scheduler
- Controller Manager
- Cloud Controller Manager(CCM)

**Components of data plane**
- Kubelet
- Kubeproxy
- Container Run Time

- In ***Docker*** simplest thing is - ***Container***
- In ***Kubernetes*** simplest thing is - ***Pod***
